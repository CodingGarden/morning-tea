## serve

> This will serve the app on port 8080.

```sh
vue-cli-service serve
```

## build

> This will build the app to the dist directory.

```sh
vue-cli-service build
```

## lint

> This will check for linting errors in the src directory.

```sh
vue-cli-service lint
```
